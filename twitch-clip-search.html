One-time setup — takes about 3 minutes.

1. Go to dev.twitch.tv/console/apps/create
2. Name the app anything (e.g. ClipFinder), set OAuth redirect to http://localhost, category: Other
3. Click Manage → copy your Client ID → click New Secret → copy it
4. Paste both below and hit Save & Connect

Credentials are saved in your browser only (localStorage). No server ever receives them. The Client Secret is used once to fetch a read-only app token and is not stored in memory after that.
