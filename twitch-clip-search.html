One-time setup — takes about 3 minutes.

1. Go to dev.twitch.tv/console/apps/create
2. Name it anything (e.g. ClipFinder), OAuth redirect → http://localhost, category → Other
3. Click Manage → copy Client ID → click New Secret → copy it
4. Paste both below and click Connect

Credentials are stored in your browser only. Your Client Secret is used once to fetch a read-only token and is never stored in memory after that.
